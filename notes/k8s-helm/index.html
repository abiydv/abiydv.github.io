K8s Helm Charts  sandbox 

15.4.24 4.4.25 / Notes / 1 min

Package manager for all the manifests that need to be deployed to a cluster

Components

  • chart.yaml - metadata about the chart
  • values.yaml - values for variables exposed by the chart author
  • templates dir - yaml manifest that define the rsources/objects to be deployed, variables are replaces with values as specified in values.yaml

Search charts called database in default repository helm seach hub database

Add a custom repository helm add repo myrep https://myrepo.io

Install myapp from myrepo on a cluster, and override values for param1, param2 helm install myapp myrepo/app --set param1=value1 --set param2=value2

Alternatives - [[kustomize]]
